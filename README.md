# Toroidal Pathfinding

Находит пути на тороидальной карте (края заворачиваются).

## Использование

```bash
cargo run
```

Введите карту через stdin с символами `#` (стены), ` ` (пусто), `i` (начало), `O` (конец). Путь отмечается точками `.`.

### Способы ввода карты

1. **Интерактивный ввод:**
```bash
cargo run
```
Введите карту построчно, затем `Ctrl+Z` (Windows) или `Ctrl+D` (Linux/Mac)

2. **Из файла:**
```bash
# Linux/Mac:
cargo run < maps/simple.txt

# Windows PowerShell:
Get-Content maps/simple.txt | cargo run
```

3. **Пайп:**
```bash
echo "##    #
#  #i #
#  O## 
   #   " | cargo run
```

## Примеры карт

### Простая карта (maps/simple.txt)
```
##    #
#  #i #
#  O## 
   #   
```

### Тороидальная карта (maps/toroidal.txt)
```
#i#  #
#  #  #
#  #  #
#  #  #
#  #O#
```

### Лабиринт (maps/maze.txt)
```
########
#i     #
# ##### #
# #   # #
# # # # #
# # # # #
#     O#
########
```

### Маленькая карта (maps/small.txt)
```
#i#
# #
#O#
```

## Формат карты

- `#` - стены (непроходимые)
- ` ` (пробел) - пустые клетки
- `i` - начальная позиция (должна быть ровно одна)
- `O` - конечная позиция (должна быть ровно одна)
- `.` - найденный путь (выводится программой)
